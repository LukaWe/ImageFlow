Preferences are stored locally (browser.storage.local) and apply instantly.

Reddit access: this extension uses your existing reddit.com login session — just stay logged in to Reddit in Firefox. No app registration or API key is needed. If Reddit changes its policies an extension update may be required. To stay well-behaved it self-limits to ~30 requests/min.

Data saver mode
Disables lookahead prefetch, defers thumbnail strips, and prefers smaller preview.redd.it variants (with a per-image “HD” button to load full resolution).
Image cache cap
Max disk used by the CDN image cache (Cache Storage) before oldest-first LRU eviction. Image caching never counts against Reddit's API limit.
