How this extension signs in to Reddit

This extension relies on your Reddit login session. If Reddit's policies change or this stops working, an update may be required.

In plain terms:

  • Just make sure you're logged in to reddit.com in Firefox. That's it — no app registration, no API keys.
  • The extension makes the same requests reddit.com already makes for you, using your own session — it only shows content you could already see by browsing normally.
  • To stay well-behaved it limits itself to ~30 requests/minute and caches results.

Open any reddit.com/r/<subreddit> page and click the floating Gallery button, or use the toolbar icon.
